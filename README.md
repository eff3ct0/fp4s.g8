# FP4S (g8)

A [Giter8](https://www.foundweekends.org/giter8) template for Scala Functional Programming Projects.

## Creating a new project from this template

Using sbt (0.13.13+) just do

```bash
sbt new eff3ct0/fp4s.g8
```
